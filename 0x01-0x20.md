# 0x0B.RSA

>   在一次RSA密钥对生成中，假设p=473398607161，q=4511491，e=17
>   求解出d作为flga提交

开始有正常的题了。

需要gmpy2的库，已知pq和e，求d：

e,d,phi满足：

-   1 < d < phi

-   E＊D mod phi ＝ 1

这里简单说下gmpy2：

-   mpz用于初始化大整数
-   invert用于求逆元
    -   E＊D mod phi ＝ 1
    -   那么`D = gmpy2.invert(E, phi)`

```python
import gmpy2

p = gmpy2.mpz(473398607161)
q = gmpy2.mpz(4511491)
e = gmpy2.mpz(17)
phi = (p - 1) * (q - 1)
d = gmpy2.invert(e, phi)
print (d)
```
